# 1D Minecraft on C++ ⛏️

Минималистичный текстовый клон легендарной песочницы Minecraft, реализованный в одномерном пространстве (1D) на чистом C++. Весь мир представляет собой горизонтальную цепочку блоков, где вы можете перемещаться, копать шахты под себя и строить башни.

Проект создан на C++ (который лидирует в статистике разработки подобных игровых систем с долей **33.0%**).

---

## 🔥 Основные возможности
* **Генерация мира**: Автоматическое распределение слоев атмосферы (воздуха), плодородной земли, твердого камня и неразрушимого бедрока на границах.
* **Честная гравитация**: Персонаж мгновенно падает вниз, если под его ногами разрушается блок воздуха.
* **Механика строительства**: Возможность ставить блоки прямо под себя, чтобы возводить защитные вышки или мосты.
* **Инвентарь**: Реализована система сбора, хранения и последующей установки добытых ресурсов.

---

## 🕹️ Управление в игре

После запуска игры управление персонажем (`P`) осуществляется через консольный ввод (нажмите клавишу + `Enter`):

| Клавиша | Действие |
| :---: | :--- |
| **A** | Движение влево |
| **D** | Движение вправо |
| **Q** | Разрушить (добыть) блок прямо под собой |
| **E** | Поставить блок из инвентаря под себя (строительство) |
| **X** | Сохранить прогресс и выйти из игры |

---

## 🧱 Обозначения объектов

* `P` — Игрок (Player)
* ` ` — Воздух (Air)
* `.` — Земля (Dirt)
* `#` — Камень (Stone)
* `X` — Бедрок / Коренная порода (Bedrock)

## 🛠️ Сборка и запуск проекта

Проект не требует сторонних графических библиотек (вроде SFML или SDL) и запускается в любой стандартной консоли.

### Способ 1: Сборка через Терминал (GCC / G++)
Перейдите в папку с файлами проекта и выполните команду:
```bash
g++ main.cpp -o 1d_minecraft
./1d_minecraft
```

### Способ 2: Сборка в IDE
1. Создайте новый проект C++ в любой удобной среде (**Visual Studio, CLion, Code::Blocks**).
2. Перенесите в папку проекта три файла: `Block.h`, `World.h` и `main.cpp`.
3. Нажмите кнопку **Run / Запуск**.
